# Hamilton & Reid LLP — AI Business Profile

This repository contains a fully structured, LLM-ready business profile for Hamilton & Reid LLP, a boutique business law firm specializing in commercial contracts, corporate governance, and regulatory compliance.

Files in this repo are optimized for ingestion by modern LLMs (ChatGPT, Gemini, Claude, Perplexity) and follow strict markdown & metadata conventions.

## Files
- `business-profile.md` — Top-level firm summary
- `services.md` — Full list of legal services
- `practice-areas.md` — Detailed business law specialties
- `pricing.md` — Pricing & billing structure
- `policies.md` — Service policies & terms
- `faqs.md` — Frequently asked questions
- `metadata.json` — Aggregate firm metadata
- `schema.yaml` — Field definitions for metadata
- `QA_CHECKLIST.md` — Documentation QA checklist
- `before-after.md` — Example cleanup transformation
- `diagrams.md` — Mermaid diagrams

## Purpose
This project demonstrates:
- Expert Markdown formatting and chunking
- JSON/YAML metadata modeling
- Cross-linked documentation structure
- LLM-friendly writing and noise removal
- Information architecture for professional services
